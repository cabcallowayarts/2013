After going through the last chapter, you've learned how to create a website. While this is cool, it doesn't look very good, just a bunch of black text on a white background with some images to break it up every once in a while. However, when you go on a normal website, you'll see much more variation than that. If you go on an average webpage you can see dozens of colors, fonts, text sizes, and elements of every shape and size that you can imagine.

This is the section that you will get to learn how to do that. The language that you will be using is CSS, which is short for 'Cascading Style Sheets'. CSS is completely different than HTML in how it works, and what it does, but once you learn a few key principles, the rest becomes much easier to learn. To get started, choose "Beginning CSS".
