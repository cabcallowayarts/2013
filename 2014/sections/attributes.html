Attributes

If you looked at all of the examples in the last section, you will notice that a few of them had some other content in the tag besides its type of element. This bit of text is called an attribute, and it contained within the tag with this format.

<p id="para"> Hello World </p>

This is an important concept once you get to CSS, and even in some parts of HTML, so we'll have another list of attributes that you can use across all elements. This list will be considerably shorter.

  • id: identification, used for singling out elements based on name
    • Example:
      <p id="approved">Some text</p>
      <p id="denied">Some corrupted text</p>
  • class: same things as id, but used differently later on
    • Example:
      <p class="approved">Some text</p>
      <p class="denied">Some corrupted text</p>
  • href/src: source, used for links (see next chapter)
    • Example:
      <a href="http://www.google.com">Some text</p>
      <img src="image.jpg" />
