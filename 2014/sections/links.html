Links

In the last section, we mentioned links a little bit, but they were not explained very much. This is one of the final things you need to learn before moving on to CSS.

A link is any connection with anything outside of you're current page. This can be another website, an image, or another file you need to connect to the current one. This is accomplished through three different means: the href and src attributes, and the link tag.

To start off, we'll talk about the href attribute. This is used with the a tag, as you may have seen. This allows you to create a link on your page to another page. For example, this code:

<a href="http://www.google.com">
Google
</a>

will send you to Google's homepage. This becomes quite useful when you're making menus to navigate to multiple pages on your site. However, you have to be careful, as the folder you set up in the very beginning is beginning to become useful.

In web developer terms, the folder, which contains your "index.html", is called a directory. In a directory are files and other "sub-directories". If you do not have a "http://" prefix on the url that you provide, it will look for a file inside your directory. If you do not have that file, it will send you to a 404 page.

Luckily, this can actually be pretty useful. For example, if you have a file in the same folder called "test.html", to link to it, you just use this:

<a href="test.html">Test</a>

Also, if you have another folder, or sub-directory, within that folder, you can access it too. If you moved the test.html into a folder within you're main folder, and called the folder "files", to get to test.html, you would use this code:

<a href="files/test.html">Test</a>

These same concepts also carry across with all other types of links. The next method we will be talking about is the src attribute. It works the same as the href attribute, but it works for images. For example, if you wanted to link an image called "cat.jpg" that you put in the "files" folder mentioned earlier, then you would put this into your code:

<img src="files/cat.jpg" />

The last type of link that we will be learning about is unique, but uses the href attribute in its process. This is the link element, and it goes into the head tag, not the body. It is used to link other files to your current one. The main application of this is to attach css files, but we will learn about that next section. Until then, here is an example. It will be fully explained in the CSS section.

<link href="files/index.css" >
