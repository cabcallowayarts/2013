More Elements

Now that you have the basics of HTML down, you can start to learn some more of the elements that you will be using when making websites. Unfortunately, there are so many that we can't list them all here. To see a full list, you can visit this site. If you want to see what any of these look like, you can copy the examples into the HTML section of the compiler.

  • p: paragraph element, good for segments of text
    • Example:
      <p>This is a small segment of text. But not too small.</p>
  • div: divider element, good for large areas; carrying other elements
    • Example:
      <div>
      <p>This div element contains multiple paragraphs</p>
      <p>This is a small segment of text. But not too small.</p>
      </div>
  • b: bold element, makes text bold
    • Example:
      <b>I'm feeling pretty bold</b>
  • i: italic element, makes text italic
    • Example:
      <i>I'm feeling pretty slanted</i>
  • h1-h6: header elements, from largest to smallest
    • Example:
      <h1>I'm very big</h1>
      <h2>I'm very decently sized</h2>
      <h3>I'm average</h3>
      <h4>I'm a bit small</h4>
      <h5>Woah, you guys are pretty big</h5>
      <h6>Aw, come one, why am I so tiny?</h6>
  • span: like p, but for smaller pieces of text
    • Example:
      <span>To be, or not to be?</span>
  • a: link element, makes links to other website
    • Example:
      <a href="http://www.google.com">Don't worry, we'll learn about it later</i>
  • ul/li: list element, and list segment
    • Example:
      <ul>
      <li>Hello</li>
      <li>World</li>
      </ul>
  • img: image element, allows you to add images (this one's a bit weird; it will be taught later)
    • Example:
      <img src="image.jpg" />
