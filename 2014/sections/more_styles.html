More Styles

Now that you understand how CSS works, it's time that you learn some of the styles so you can start to use it. This section will be less of a lecture and more of a list. There will be a list of 20-30 styles that can get you started in using CSS effectively. If you are curious, there will be a link in the references section that will contain a complete list of CSS styles. Anyways, without further ado, here is the list:

  • color: change the color of the text
    • Possible Values: Most colors, as text
    • Examples:
      color: red;
      color: green;
      color: blue;
  • background-color: change the color of the background
    • Possible Values: Most colors, as text, same as color
    • Examples:
      background-color: yellow;
      background-color: purple;
      background-color: black;
  • font-size: set the size of the font
    • Possible Values: px and em's
    • Examples:
      font-size: 10px;
      font-size: 1.5em;
      font-size: 72px;
  • width: set the width of the element
    • Possible Values: px and %
    • Examples:
      width: 10px;
      width: 50%;
      width: 17px;
  • height: set the height of the element
    • Possible Values: px and %
    • Examples:
      height: 10px;
      height: 50%;
      height: 18px;
  • margin: set the margin of the element (also comes in margin-left, margin-top, margin-right, and margin-bottom)
    • Possible Values: px and %
    • Examples:
      margin: 1%;
      margin: 10px;
      margin: 15%;
  • padding: set the padding of the element (also comes in padding-left, padding-top, padding-right, and padding-bottom)
    • Possible Values: px and %
    • Examples:
      padding: 1%;
      padding: 10px;
      padding: 15%;
  • display: determines whether element is inline(stays on given line), or block(has its own line)
    • Possible Values: block and inline
    • Examples:
      display: inline;
      display: block;
  • position: determines whether element is automatically placed, or can move around freely
    • Possible Values: absolute(move around freely), fixed(like absolute, but not affected by scrolling), and static(placed automatically)
    • Examples:
      position: absolute;
      display: static;
  • left/right: determines how far from the left/right an element is (only works with absolute or fixed position)
    • Possible Values: px and %
    • Examples:
      left: 50%;
      right: 10px;
  • top/bottom: determines how far from the top/bottom an element is (only works with absolute or fixed position)
    • Possible Values: px and %
    • Examples:
      top: 50%;
      bottom: 10px;
